# Система управления библиотекой

Консольное приложение на языке C++ для управления библиотекой.
Программа позволяет хранить информацию о книгах и пользователях, выполнять выдачу и возврат книг, а также сохранять и загружать данные из файла.

---

## Функциональные возможности

- Просмотр всех книг
- Просмотр всех пользователей
- Добавление новой книги
- Регистрация пользователя
- Выдача книги пользователю
- Возврат книги в библиотеку
- Поиск книги по ISBN
- Просмотр профиля пользователя
- Сохранение данных в файл
- Автоматическая загрузка данных при запуске программы

---

## Структура проекта

- `src/` — исходные файлы программы
- `data/library_data.txt` — файл с данными библиотеки
- `CMakeLists.txt` — файл конфигурации сборки
- `README.md` — описание проекта

---

## Требования

- Компилятор с поддержкой **C++17**
- **CMake версии 3.10** или выше

---

## Сборка проекта

Сборка выполняется из корня проекта.

```bash
mkdir build
cd build
cmake ..
make
Программа использует файл данных по относительному пути:
data/library_data.txt
Поэтому программа должна запускаться из корня проекта.
